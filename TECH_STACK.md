# Technology Stack

## Frontend
- **Framework**: Next.js 14.0.4
- **Language**: TypeScript 5.3
- **Styling**: TailwindCSS 3.3.6
- **State**: React Hooks, Context
- **Icons**: Lucide React
- **Dependencies**: Lucide-react, Framer-motion, Axios, React-hook-form

## Backend
- **Framework**: FastAPI 0.104.1
- **Language**: Python 3.10+
- **ORM**: SQLAlchemy 2.0.23 (Async)
- **Driver**: Asyncpg
- **Validation**: Pydantic 2.5.0
- **Auth**: JWT (Python-jose, Passlib)

## Database
- **Primary**: PostgreSQL
- **Cache**: Redis 5.0.1 (Python client)

## Infrastructure
- **Deploy**: Docker / Manual
- **CI/CD**: Not configured

## Integrations
- **Payments**: Stripe (stripe==5.4.0)
- **AI**: OpenAI, Anthropic, Google, Mistral, Grok (via HTTP/Client)
